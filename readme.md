# Задача: 258

Дано целое число `num`. Повторно складывайте все его цифры, пока результат не станет однозначным, и верните его.

## Пример

**Input:**
```
num = 38
```

**Output:**
```
2
```

**Explanation:**
Процесс вычислений следующий:
```
38 --> 3 + 8 --> 11
11 --> 1 + 1 --> 2
```
Так как результат (2) состоит из одной цифры, возвращаем его.
